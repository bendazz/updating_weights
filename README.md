## Linear Regression Visualizer

A minimal frontend app that draws a simple linear model with two inputs and one output (no hidden layer) using SVG.

### Quick Start

- Open `index.html` directly in your browser, or serve the folder with a simple HTTP server.

#### Option A: Open directly
- Double-click `index.html` or drag it into your browser.

#### Option B: Serve locally (Python)
```bash
python3 -m http.server 8080
# then visit http://localhost:8080/
```

### Files
- `index.html`: App entry point
- `style.css`: Basic styling for layout and visuals
- `script.js`: SVG rendering of the model

Feel free to adjust positions and labels in `script.js` to fit your lesson.
